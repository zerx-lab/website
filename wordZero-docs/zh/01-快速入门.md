# 快速入门指南

本指南将帮助您快速开始使用 WordZero。WordZero 是一个高性能的 Go 语言库，用于创建、读取和操作 Microsoft Word 文档（.docx）。

## 安装

### 环境要求

- Go 1.19 或更高版本

### 通过 Go Modules 安装

```bash
go get github.com/zerx-lab/wordZero
```

## 创建第一个文档

让我们创建一个简单的 Word 文档：

```go
package main

import (
    "log"
    "github.com/zerx-lab/wordZero/pkg/document"
)

func main() {
    // 创建新文档
    doc := document.New()

    // 添加段落
    doc.AddParagraph("你好，WordZero！")

    // 保存文档
    if err := doc.Save("hello.docx"); err != nil {
        log.Fatal(err)
    }
}
```

## 基本操作

### 创建文档

```go
// 创建一个新的空文档
doc := document.New()

// 打开已有的文档
doc, err := document.Open("existing.docx")
if err != nil {
    log.Fatal(err)
}
```

### 添加内容

```go
// 添加简单段落
doc.AddParagraph("这是一个段落。")

// 添加标题
doc.AddHeadingParagraph("第一章：介绍", 1)

// 添加格式化文本
format := &document.TextFormat{
    Bold:      true,
    FontSize:  14,        // int 类型（磅）
    FontColor: "FF0000",  // 红色
}
doc.AddFormattedParagraph("加粗的红色文本", format)
```

### TextFormat 结构体

```go
type TextFormat struct {
    Bold       bool   // 加粗
    Italic     bool   // 斜体
    FontSize   int    // 字号（磅）
    FontColor  string // 十六进制颜色代码（如 "FF0000"）
    FontFamily string // 字体名称（如 "Arial"）
    FontName   string // 字体名称别名
    Underline  bool   // 下划线
    Strike     bool   // 删除线
    Highlight  string // 高亮颜色
}
```

### 在一个段落中添加多种格式的文本

```go
para := doc.AddParagraph("")
para.AddFormattedText("普通文本，", nil)
para.AddFormattedText("加粗文本，", &document.TextFormat{Bold: true})
para.AddFormattedText("斜体文本。", &document.TextFormat{Italic: true})
```

### 保存文档

```go
// 保存到文件
err := doc.Save("output.docx")

// 获取字节数据（用于 HTTP 响应等）
data, err := doc.ToBytes()
```

## 使用样式

WordZero 内置了 18 种预定义样式：

```go
import "github.com/zerx-lab/wordZero/pkg/style"

// 应用标题样式
para := doc.AddParagraph("我的标题")
para.SetStyle(style.StyleHeading1)

// 可用的预定义样式：
// - style.StyleHeading1 到 style.StyleHeading9（标题1-9）
// - style.StyleTitle（文档标题）
// - style.StyleSubtitle（副标题）
// - style.StyleNormal（正文）
// - style.StyleQuote（引用）
// - style.StyleListParagraph（列表段落）
// - style.StyleCodeBlock（代码块）
// - style.StyleEmphasis（强调-斜体）
// - style.StyleStrong（加粗）
// - style.StyleCodeChar（行内代码）
```

## 创建表格

```go
// 使用 TableConfig 创建表格
table, err := doc.AddTable(&document.TableConfig{
    Rows:  3,
    Cols:  3,
    Width: 8000, // 宽度（磅）
    Data: [][]string{
        {"表头 1", "表头 2", "表头 3"},
        {"数据 1", "数据 2", "数据 3"},
        {"数据 4", "数据 5", "数据 6"},
    },
})
if err != nil {
    log.Fatal(err)
}

// 或者使用 CreateTable
table, err := doc.CreateTable(&document.TableConfig{
    Rows:  3,
    Cols:  4,
    Width: 8000,
})
```

## 页面设置

```go
// 设置页面大小为 A4
doc.SetPageSize(document.PageSizeA4)

// 设置页面方向为横向
doc.SetPageOrientation(document.OrientationLandscape)

// 设置边距（上、右、下、左，float64 类型）
doc.SetPageMargins(25.4, 25.4, 25.4, 25.4)
```

## 添加图片

```go
// 使用尺寸配置从文件添加图片
imgConfig := &document.ImageConfig{
    Size: &document.ImageSize{
        Width:  100, // 宽度（毫米）
        Height: 80,  // 高度（毫米）
    },
}
info, err := doc.AddImageFromFile("photo.jpg", imgConfig)
if err != nil {
    log.Fatal(err)
}
```
