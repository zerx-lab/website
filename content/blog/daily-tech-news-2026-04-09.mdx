### LittleSnitch 登陆 Linux：25 年后首次跨平台，免费且部分开源 ⭐⭐⭐⭐

**核心要点：**

- **2026 年 4 月 8 日**，奥地利公司 Objective Development Software 发布 **Little Snitch for Linux**，这是这款 macOS 网络监控工具诞生 25 年以来首次移植到其他平台。
- Linux 版完全**免费**，且将保持永久免费；eBPF 内核组件与 Web UI 均已在 GitHub 开源（GPL v2），后端 daemon 为免费闭源授权。
- 技术栈：使用 **eBPF** 进行内核级流量拦截，**Rust** 编写主应用，**Web 界面**可跨平台访问（包括远程监控 Linux 服务器）。
- 实测发现：Ubuntu 系统每周约有 9 个系统进程联网，而 macOS 同期超过 100 个——Linux 系统在"默认出站流量"上远比 macOS 克制。
- 开发动机：creator 坦言是"政治因素"推动——近期欧洲政府大力推动减少对外国软件依赖，Linux 成为明确候选，自然催生对 Linux 隐私工具的强烈需求。

**技术解读：**

选择 eBPF 而非传统内核模块是一个经典的现代 Linux 内核编程决策：eBPF 程序运行在沙箱中，不会导致内核崩溃，但资源限制也意味着它无法完全阻止"蓄意逃逸"的恶意软件。因此 Linux 版定位明确为**隐私工具**（观察和控制日常应用的网络行为）而非**安全工具**（对抗高级威胁），这与 macOS 版的定位有本质差异。Web 界面的选择则让 homelab 用户可以直接从其他设备监控 Linux 服务器，是对 CLI 工具的重要补充。

**开发者行动建议：**

- 在 [obdev.at/products/littlesnitch-linux](https://obdev.at/products/littlesnitch-linux/download.html) 下载，重点适用场景：homelab 服务器、Docker 主机出站流量审计
- 注意：它是隐私工具，不应替代 nftables/iptables 作为安全控制层
